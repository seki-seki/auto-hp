事業内容

不動産事業

独自のネットワークと不動産の知識で賃貸から不動産投資まで幅広い範囲で対応致します。お客様にとって最適な価値の創造を常に考え提案します。

建築工事設計

最高のデザインと独自性を兼ね備えた建築設計を提供します。

看板・広告事業

独自のデザインで最適化し、唯一無二の広告を提案します。

保険事業

生命保険募集、損害保険代理店事業でお客様の資産と生活を守ります。

不動産コンサルタント

住宅地・別荘地の開発造成など、専門的なコンサルティングを提供します。

総合サポート

不動産から保険まで、お客様の人生設計を総合的にサポートいたします。専門知識と経験で、最適なソリューションを提供します。

MESSAGE

代表メッセージ

私たちの使命は「価値を創造し、社会に還元すること」です。

激しく変化する現代社会において、単にモノやサービスを提供するだけではなく、人々の期待を超える「新たな価値」を生み出すことこそが企業に求められる本質的な役割だと私たちは考えています。

創業以来、私たちは常に「本質は何か」「誰のためにあるのか」を問い続け、ユーザー視点と未来志向を大切にしながら事業を展開してまいりました。

代表社員　井上真意

FEATURES

選ばれる理由

🎨

最高のデザイン

Design × Place - 最高のデザインと独自性で、お客様の期待を超える価値を創造します。デジタルでは再現できない独自性とメディアミックスを実現します。

自由な創造力

Creation - 固定観念にとらわれない自由な創造力で、新しい価値を生み出します。お客様の希望を形にし、唯一無二の空間とサービスを提供します。

🤝

確かな実績

長年培ってきた経験と実績で、お客様に最適なソリューションを提供します。不動産から建築、広告、保険まで、トータルサポートが可能です。
