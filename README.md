# Gaia LLC Web Concepts

本リポジトリには、Atlassian Design Systemを参考にデザインしたGaia LLC向けのウェブページ案が3パターン収録されています。`pages/` ディレクトリ内のHTMLをブラウザで開くことで、それぞれのデザインを確認できます。

- `pages/pattern-one.html` — ブルーのグラデーションを基調にしたフラッグシップ案。
- `pages/pattern-two.html` — サイドバー構成のカード型レイアウト案。
- `pages/pattern-three.html` — ダークヒーローを採用した没入型レイアウト案。

各ページのヘッダーから他パターンに遷移できます。最終的に採用するデザインを選定する際は、該当ファイルを単独で提供してください。
